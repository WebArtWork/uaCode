Політика конфіденційності

Ця політика конфіденційності описує, як ми збираємо, використовуємо та захищаємо вашу особисту інформацію під час використання нашого вебсайту або додатку, створеного на основі UA Code.

1. Збір інформації

Ми можемо збирати мінімальні дані, такі як ім'я користувача або відповідь на завдання, з метою освітнього процесу. Ми не зберігаємо ніяких персональних або конфіденційних даних без вашої згоди.

2. Використання інформації

Вся інформація використовується виключно для покращення навчального досвіду, надання зворотного зв’язку та статистики прогресу учня.

3. Безпека

Ми вживаємо всіх необхідних заходів для захисту вашої інформації від несанкціонованого доступу чи розголошення.

4. Права користувача

Користувач має право звернутись до нас з вимогою видалення своїх навчальних даних. Для цього зв’яжіться з нами через форму зворотного зв’язку.

5. Зміни до політики

Ми можемо час від часу оновлювати цю політику. Будь-які зміни будуть опубліковані на цій сторінці з оновленою датою.
